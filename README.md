# 🏡 Property Guys Estate

A modern and responsive real estate website built using HTML, CSS, and JavaScript. The website showcases premium properties with an attractive design and a user-friendly interface. It is designed to work smoothly on desktops, tablets, and mobile devices.

## ✨ Features

- Responsive design for all screen sizes
- Modern homepage with hero section
- Featured property cards
- Property images with details
- Contact section
- Click-to-call phone number
- Smooth navigation
- Clean and organized code
- Interactive functionality using JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome

## 📱 Responsive Design

The website is optimized for:
- Desktop
- Tablet
- Mobile
